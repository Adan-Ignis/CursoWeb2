1. ¿Qué es SQL?
A) Un sistema operativo
B) Un lenguaje para gestionar bases de datos
C) Un editor de texto
D) Un navegador web

Respuesta: B

2. ¿Qué comando se utiliza para consultar datos de una tabla?
A) GET
B) SELECT
C) SHOW
D) VIEW

Respuesta: B

3. ¿Qué comando se usa para insertar datos en una tabla?
A) ADD
B) INSERT INTO
C) CREATE
D) UPDATE

Respuesta: B

4. ¿Qué comando se usa para modificar datos existentes?
A) CHANGE
B) MODIFY
C) UPDATE
D) ALTER DATA

Respuesta: C

5. ¿Qué comando elimina registros de una tabla?
A) REMOVE
B) DELETE
C) DROP
D) CLEAR

Respuesta: B

6. ¿Qué comando se usa para crear una tabla?
A) CREATE TABLE
B) MAKE TABLE
C) NEW TABLE
D) BUILD TABLE

Respuesta: A

7. ¿Qué comando elimina una tabla completa?
A) DELETE TABLE
B) DROP TABLE
C) REMOVE TABLE
D) ERASE TABLE

Respuesta: B

8. ¿Qué cláusula se usa para filtrar registros?
A) FILTER
B) HAVING
C) WHERE
D) ORDER

Respuesta: C

9. ¿Qué cláusula se usa para ordenar resultados?
A) ORDER BY
B) SORT BY
C) GROUP BY
D) ALIGN BY

Respuesta: A

10. ¿Qué operador se usa para buscar coincidencias parciales?
A) =
B) LIKE
C) MATCH
D) PART

Respuesta: B

11. ¿Qué función cuenta registros?
A) SUM()
B) TOTAL()
C) COUNT()
D) NUMBER()

Respuesta: C

12. ¿Qué cláusula agrupa registros con valores iguales?
A) GROUP BY
B) ORDER BY
C) WHERE

13. ¿Qué palabra clave se usa para evitar duplicados?
A) UNIQUE
B) DISTINCT
C) DIFFERENT
D) ONLY

Respuesta: B

14. ¿Qué es una clave primaria?
A) Un campo que identifica de forma única cada registro
B) Un campo opcional

B) MAKE VIEW
C) NEW VIEW

B) UPDATE TABLE
