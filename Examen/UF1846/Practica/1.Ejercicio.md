Dado el siguiente sistema:

    - Un navegador muestra una web
    - Un servidor procesa peticiones
    - Una base de datos almacena usuarios

Responde:

    1. ¿Qué parte es el frontend?

    Respuesta:
    
        "Un navegador muestra una web"

    2. ¿Qué parte es el backend?

    Respuesta:
    
        "Un servidor procesa peticiones" y "Una base de datos almacena usuarios"

    3. ¿Es una arquitectura distribuida? ¿Por qué?

    Respuesta:

        Sí se puede considerar una arquitectura distribuida, porque cada parte cumple una función distinta y puede estar cada una en un equipo distinta.

        Un ordenador de poca potencia puede encargarse de un navegador que muestra una web, un servidor para procesar las peticiones y otro servidor para la base de datos.

        Es importante mencionar que cada servidor tiene unos componentes diferentes para obtener una mayor potencia en su respectiva area con un menor consumo energetico.

        A pesar de lo mencionado un solo ordenador puede ejecutar todo esto al mismo tiempo.
