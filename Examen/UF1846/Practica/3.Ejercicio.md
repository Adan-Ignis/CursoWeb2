Diseña una API para una aplicación de tareas (to-do list).

Define al menos 4 endpoints:

    - Obtener tareas

		Request:

			GET /tareas

		Response:

			[
				{
					"id": 1,
					"titulo": "Estudiar API",
					"descripcion": "Repasar endpoints REST",
					"completada": false
				}
			]

    - Crear tarea

		Request:

			POST /tareas

		Body (JSON):

			{
				"titulo": "Hacer ejercicio",
				"descripcion": "Correr 30 minutos"
			}

		Response:

			{
				"id": 2,
				"titulo": "Hacer ejercicio",
				"descripcion": "Correr 30 minutos",
				"completada": false
			}

    - Actualizar tarea

		Request:

			PUT /tareas/{id}

		Ejemplo:

			PUT /tareas/2

		Body:

			{
				"titulo": "Hacer ejercicio",
				"descripcion": "Correr 45 minutos",
				"completada": true
			}

		Response:

			{
				"mensaje": "Tarea actualizada correctamente"
			}

	- Eliminar tarea

		Request:

			DELETE /tareas/{id}

		Ejemplo:

			DELETE /tareas/2

		Response:

			{
				"mensaje": "Tarea eliminada correctamente"
			}
