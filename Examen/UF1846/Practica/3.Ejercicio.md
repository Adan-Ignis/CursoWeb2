Diseña una API para una aplicación de tareas (to-do list).

Define al menos 4 endpoints:

    - Obtener tareas

Request

GET /tareas

Response

[
  {
    "id": 1,
    "titulo": "Estudiar API",
    "descripcion": "Repasar endpoints REST",
    "completada": false
  }
]

    - Crear tarea
Request

POST /tareas

Body (JSON)

{
  "titulo": "Hacer ejercicio",
  "descripcion": "Correr 30 minutos"
}

Response

{
  "id": 2,
  "titulo": "Hacer ejercicio",
  "descripcion": "Correr 30 minutos",
  "completada": false
}


    - Actualizar tarea

Request

PUT /tareas/{id}

Ejemplo

PUT /tareas/2

Body

{
  "titulo": "Hacer ejercicio",
  "descripcion": "Correr 45 minutos",
  "completada": true
}

Response

{
  "mensaje": "Tarea actualizada correctamente"
}

    - Eliminar tarea

Request

DELETE /tareas/{id}

Ejemplo

DELETE /tareas/2

Response

{
  "mensaje": "Tarea eliminada correctamente"
}

from flask import Flask, request, jsonify

app = Flask(__name__)

tareas = []

@app.route('/tareas', methods=['GET'])
def obtener_tareas():
    return jsonify(tareas)

@app.route('/tareas', methods=['POST'])
def crear_tarea():
    data = request.json
    tarea = {
        "id": len(tareas) + 1,
        "titulo": data["titulo"],
        "descripcion": data["descripcion"],
        "completada": False
    }
    tareas.append(tarea)
    return jsonify(tarea), 201
