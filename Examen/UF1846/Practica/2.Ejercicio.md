Tienes esta API:

    GET /usuarios
    POST /usuarios
    GET /usuarios/5
    DELETE /usuarios/5

Responde:

    1. ¿Qué hace cada endpoint?

    Respuesta:

        - GET /usuarios
        
            Obtiene la lista de todos los usuarios
            
        - POST /usuarios
        
            Crea un nuevo usuario con los datos que se envían en la petición
            
        - GET /usuarios/5
        
            Obtiene la información del usuario con ID 5
            
        - DELETE /usuarios/5
        
            Elimina el usuario con ID 5

    2. ¿Cuál usarías para crear un usuario?

    Respuesta:

        POST /usuarios

    3. ¿Cuál usarías para eliminarlo?

    Respuesta:

        DELETE /usuarios/5
