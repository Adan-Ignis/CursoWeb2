1. Una empresa quiere desplegar una aplicación interna solo accesible por sus
empleados.
¿Qué tipo de entorno debe utilizar y qué medidas básicas de seguridad
implementarías?

Implementaria una intranet con las siguientes caracteristicas:

- El despliegue debe realizarse en un servidor (local o en la nube) que no esté indexado públicamente (que no aparezca en las busquedas de un buscador).

Para que los empleados accedan se puede usar una VPN (Virtual Private Network) o una ZTNA (Zero Trust Network Access). Esta ultima es más seguro porque por defecto no se confía en nadie por defecto, vertificando la identidad de cada usuario y dispositivo antes de conceder acceso a la aplicación específica.

- Las medidas básicas de seguridad debe tener las siguientes capas:

    MFA (Autenticación de doble factor): Esta es la medida más crítica. Para acceder se debe revisar el telefono.

    RBAC (Control de Acceso Basado en Roles): No todos los empleados deben ver todo, solamente la parte que le corresponde.

    SSO (Single Sign-On): Permite usar credenciales corporativas para entrar, facilitando la baja de acceso si el empleado deja la empresa.

- Seguridad en la comunicación:

    HTTPS: Aunque la aplicación sea interna, el tráfico debe ir cifrado para evitar ataques (como "Man-in-the-Middle") detra de la propia oficina.

    Certificados digitales emitidos por una entidad de confianza para validad que el servidor es quien dice ser.

- Protección de infraestructura:

    WAF (Firewall de Aplicaciones Web): Para filtrar peticiones maliciosas y proteger contra ataques comunes como SQL Injection.

    Segmentación de la Red: Aisalar el servidor de la aplicación del resto de los equipos de la oficina para que, si un ordenador de escritorio se infecta, el virus no llegue fácilmente a la aplicación interna.

- Gestión y mantenimiento:

    Logging (Registro de sesión): Guardar un historial de quien entró, a qué hora y qué archivos consultó.

    Actualizaciones (Patching): Mantener tanto el sistema operativo como el framework de la aplicación actualizados para cerrar vulnerabilidades conocidas.
