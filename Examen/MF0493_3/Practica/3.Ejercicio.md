3. Tu aplicación ha sufrido una pérdida de datos crítica.
Explica cómo un sistema de backup y recuperación puede solucionar este problema.

1 - Disponibilidad de una copia íntegra (backup)

El primer paso es la existencia de una copia de los datos almacenado en una ubicación distinta a la original (siguiendo habitualmente la regla 3-2-2: tres copias, en dos soportes distintos, una de ellas fuera de línea o en la nube).

    Punto de recuperación: El sistema permite seleccionar la versión más reciente y estable de los datos antes de que ocurriera el fallo o la corrupción.

2 - El proceso de restauración

Una vez identificada la pérdida, el sistema de recuperación vuelca los datos respaldados sobre la infraestructura productiva. Este proceso soluciona el problema mediante:

    Reemplazo de datos dañados: Los archivos corruptos o eliminados se sustituyen por las versiones íntegras del backup.

    Reconstrucción de bases de datos: Se aplican logs de transacciones para asegurar que no se pierda información entre el último backup completo y el momento del fallo.

3 - Cumplimiento de los Objetivos de Tiempo y Putno (PTO y RPO)

Un buen sistema de backup se mide por dos métricas que dictan cómo se soluciona la crisis:

    RPO (Recovery Point Objective): Determina cuánta información se ha "perdido" realmente. Si haces backups cada hora, la pérdida máxima será de 60 minutos de datos.

    RTO (Recovery Time Objective): Es el tiempo que tarda la aplicación en volver a estar en línea. Un sistema eficiente reduce este tiempo al mínimo para evitar pérdidas económicas.

4 - Protección contra Ransomware y Errores Humanos

Si la pérdida de datos se debe a un ataque de cifrado (Ransomware) o a un borrado accidental por parte de un usuario:

    Inmutabilidad: Muchos sistemas modernos ofrecen backups inmutables que no pueden ser borrados ni cifrados, permitiendo recuperar la aplicación incluso si el atacante tiene credenciales de administrador.

    Versiones históricas: Permite retroceder en el tiempo a un estado anterior al error humano.
